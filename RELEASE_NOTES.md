# FlyForge v0.1.0 发布说明

> **发布日期**: 2026-06-18  
> **版本**: v0.1.0  
> **平台**: Windows x64  
> **安装包**: `FlyForge_0.1.0_x64-setup.exe`

---

## 简介

FlyForge 是一款飞轮设计与运动学仿真桌面应用，专为机械工程领域打造。基于 Tauri v2 + SolidJS + Rust 构建，提供高性能的飞轮参数设计、应力分析、动力学仿真与数据可视化功能。

---

## 核心功能

### 飞轮参数设计
- **两种飞轮类型**: 环形轮（等厚度）、锥形轮
- **几何尺寸**: 外径 Ro、内径 Ri、轮缘厚度、轮毂径、轮毂厚度
- **材料库**: 12 种预设材料（AISI 4340 合金钢、7075-T6 铝合金、Ti-6Al-4V 钛合金、T700 碳纤维、Q235 碳素钢、C45 碳素钢、42CrMo 合金钢、18Ni 马氏体时效钢、6061-T6 铝合金、QT600-3 球墨铸铁、HT250 灰铸铁、T1000 碳纤维）
- **工况参数**: 额定/最大/最小转速、安全系数（屈服/疲劳/破裂）、离散点数量
- **预设方案**: 8 种典型飞轮配置一键加载

### 飞轮截面预览
- SVG 实时渲染飞轮截面轮廓
- 滚轮缩放 + 鼠标拖拽平移
- 网格背景 + 坐标轴
- 颜色图例标注（外径/内径/轮毂）
- 重置视图按钮

### 可视化图表
- **应力分布图**: von Mises、周向应力、径向应力随半径变化曲线
- **转速曲线**: 启动/制动过程转速-时间曲线
- **能量柱状图**: 额定/最大/可用储能对比
- 十字准星跟踪 + 实时数据浮动框
- 滚轮缩放 + 拖拽平移 + 重置按钮
- 特殊极值点标注

### 计算结果
- 质量、转动惯量
- 最大应力及位置
- 屈服/疲劳安全系数
- 安全破裂转速
- 额定/可用/最大储能、比能量
- 安全评估状态（通过/未通过）

### 数据导出
- CSV 格式导出仿真数据
- JSON 格式导出完整结果
- SVG 格式导出图表
- 参数 JSON 导出
- 支持自定义保存路径

---

## 界面特性

- **黑绿科技配色**: 深色背景 + 翠绿强调色，专业工程软件风格
- **可折叠侧边栏**: 左右面板支持显示/隐藏
- **可调整布局**: 中央区分隔线拖拽调整上下比例
- **长按微调**: 数字输入框支持长按快速调整
- **扁平化图标**: 飞轮造型应用图标
- **实时计算**: 300ms 防抖自动重算

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | SolidJS + TypeScript |
| UI 样式 | Tailwind CSS |
| 构建工具 | Vite |
| 后端/桌面 | Tauri v2 (Rust) |
| 计算核心 | Rust (flyforge-core) |
| 安装包 | NSIS |

## 系统要求

- **操作系统**: Windows 10/11 (x64)
- **内存**: 4GB+

## 安装方式

下载 `FlyForge_0.1.0_x64-setup.exe`，双击运行安装向导，按提示完成安装。

## 已知问题

- 首次启动可能需要等待 WebView2 初始化
- 大离散点数量（>300）计算可能需要 1-2 秒
