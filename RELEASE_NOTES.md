## v0.2.0 新增功能

### 参数导入
- 导入面板新增"导入参数 JSON"按钮
- 支持 File System Access API 文件选择器
- 支持旧版本元素 `<input>` 回退方案
- 导入后自动触发仿真计算

### 自定义材料
- 材料选择器新增"自定义材料"选项
- 支持输入密度、屈服强度、抗拉强度、疲劳极限
- 自定义材料持久化到 localStorage，重启不丢失
- 支持删除自定义材料（自动切回内置材料）

### 图表交互增强
- 应力分布图添加屈服强度参考线（黄色虚线）
- 屈服强度超出图表范围时标注"↑"并置顶显示
- 转速曲线添加额定转速（绿色）和最大转速（红色）参考线
- Y 轴范围以实际数据为准，不因参考线值拉伸

### 布局记忆
- 侧边栏折叠状态、中央区上下比例持久化
- 飞轮类型和材料选择跨会话记忆
- 重启应用后自动恢复上次布局

### 错误提示优化
- 参数校验失败时所有输入框显示红色边框
- 几何尺寸面板顶部显示校验错误横幅

## 优化与修复

- von Mises 图例汉化为「等效应力」
- 移除顶栏和结果面板的「计算中」转圈动画
- 输入框支持手动输入小数（本地 text buffer 方案）
- 微调按钮浮点精度舍入（消除 1.29999 / 1.70000 等长尾数）
- 应力图 Y 轴范围从「屈服强度 ×1.15」修复为「数据范围 ×1.15」

## 简介

FlyForge 是一款飞轮设计与运动学仿真桌面应用，专为机械工程领域打造。基于 Tauri v2 + SolidJS + Rust 构建，提供高性能的飞轮参数设计、应力分析、动力学仿真与数据可视化功能。

---

## 核心功能

### 飞轮参数设计
- **5 种飞轮类型**: 实心圆盘、环形轮（等厚度）、锥形盘、等强度轮、多层复合轮
- **几何尺寸**: 外径 Ro、内径 Ri、轮缘厚度、轮毂径、轮毂厚度
- **材料库**: 12 种预设材料 + 自定义材料
- **工况参数**: 额定/最大/最小转速、安全系数（屈服/疲劳/破裂）、离散点数量
- **预设方案**: 8 种典型飞轮配置一键加载

### 飞轮截面预览
- SVG 实时渲染飞轮截面轮廓
- 滚轮缩放 + 鼠标拖拽平移
- 网格背景 + 坐标轴 + 颜色图例

### 可视化图表
- **应力分布图**: 等效应力、周向应力、径向应力 + 屈服强度参考线
- **转速曲线**: 启动/制动过程 + 额定/最大转速参考线
- **能量柱状图**: 额定/最大/可用储能对比
- 十字准星跟踪 + 实时数据浮动框
- 极值点标注

### 计算结果
- 质量、转动惯量、最大应力及位置
- 屈服/疲劳安全系数、安全破裂转速
- 额定/可用/最大储能、比能量
- 安全评估状态（通过/未通过）

### 数据导入/导出
- CSV / JSON / SVG / 参数 JSON 导出
- JSON 参数导入
- 支持自定义保存路径（File System Access API）

---

## 界面特性

- **黑绿科技配色**: 深色背景 + 翠绿强调色，全界面统一暗色主题
- **可折叠侧边栏**: 左右面板支持显示/隐藏（状态记忆）
- **可调整布局**: 中央区分隔线拖拽调整上下比例（比例记忆）
- **长按微调**: 数字输入框支持长按快速调整
- **实时计算**: 300ms 防抖自动重算
- **布局记忆**: 侧边栏、布局比例、飞轮类型和材料重启恢复
