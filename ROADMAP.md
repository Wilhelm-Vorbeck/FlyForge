## v0.3.0 — 扩展飞轮类型与高级分析

**主题**：完善全部 5 种飞轮类型的完整 UI 支持，增加疲劳寿命和灵敏度分析。

### 现状分析

| 飞轮类型 | Rust 核心 | UI 参数面板 | 截面预览 |
|----------|----------|-----------|---------|
| 实心圆盘 | ✅ | 通用面板 | 圆形 ❌ |
| 环形轮 | ✅ | 通用面板 | 圆形 ❌ |
| 锥形盘 | ✅ | 通用面板 | 圆形 ❌ |
| 等强度轮 | ✅ | 通用面板 | 圆形 ❌ |
| 多层复合轮 | ✅ | 通用面板 | 圆形 ❌ |

问题：所有类型的 GeometryTab 和 SectionPreview 完全相同，没有针对类型差异做适配。

### A. 几何面板适配飞轮类型
 - [ ] 实心圆盘：隐藏内径 Ri（自动设为 0）
 - [ ] 锥形盘：轮缘厚 →「轮缘厚度(外径处)」，轮毂厚 →「轮毂厚度(内径处)」
 - [ ] 等强度轮：厚度 →「中心最大厚度」，显示厚度曲线 `t(r) = t_center × Ri / r`
 - [ ] 多层复合轮：显示当前图层数，显示简化参数说明

### B. 截面预览 SVG 适配
 - [ ] 实心圆盘：填充蓝色圆盘（当前已是圆形）
 - [ ] 锥形盘：绘制梯形截面（外径处薄/内径处厚或反向）
 - [ ] 等强度轮：绘制双曲线形截面（t ∝ 1/r）
 - [ ] 多层复合轮：绘制分段阶梯形截面，颜色区分轮毂/腹板/轮缘

### C. 多层复合轮图层编辑器
 - [ ] 扩展 `FlywheelParams` 增加 `layer_configs` 字段（每层材料+厚度+外径）
 - [ ] 前/后端类型同步更新
 - [ ] 图层编辑 UI：添加/删除层，每层选择材料和厚度
 - [ ] Rust 端解析图层配置替代硬编码比例

### D. 疲劳寿命估算
 - [ ] Rust 端新增 `fatigue.rs` 模块：基于 S-N 曲线计算疲劳寿命
 - [ ] 输出：预期循环次数、等效使用年限、疲劳安全裕度
 - [ ] 结果面板新增疲劳寿命卡片

### E. 参数敏感性分析
 - [ ] 新图表类型：选择 X 轴参数（如外径 50-500mm）和 Y 轴指标（如最大应力）
 - [ ] 自动扫描参数范围生成灵敏度曲线
 - [ ] 标注数据最优点和危险区域

### F. 温度场影响（可选）
 - [ ] 温度输入字段
 - [ ] 材料热膨胀系数、屈服强度温度修正
 - [ ] 热应力 + 离心应力耦合分布图
