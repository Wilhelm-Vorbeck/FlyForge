## v0.2.0 — 功能补全与体验优化

**主题**：补全已有但未接入 UI 的功能，优化日常使用体验。

### 1. 参数导入功能
- [ ] 前端添加"导入参数"按钮（Rust 端 `import_params` 已实现，需 UI 接入）
- [ ] 支持拖拽 JSON 文件到应用窗口直接导入
- [ ] 导入后自动触发仿真计算

### 2. 自定义材料支持
- [ ] MaterialTab 添加"自定义材料"选项
- [ ] 输入密度、屈服强度、抗拉强度、疲劳极限等参数
- [ ] 自定义材料保存到本地 localStorage
- [ ] 自定义材料与预设材料统一管理

### 3. 图表交互增强
- [ ] 鼠标悬停曲线时显示精确数值 tooltip
- [ ] 应力分布图标注屈服强度参考线
- [ ] 能量柱状图添加数值标签
- [ ] 转速曲线标注额定/最大转速参考线

### 4. 错误提示优化
- [ ] 参数校验失败时在输入框旁显示红色提示（而非仅 console 报错）
- [ ] 安全评估未通过时高亮显示关键风险指标
- [ ] 添加操作引导提示（首次使用教程）

### 5. 布局记忆
- [ ] 侧边栏折叠状态持久化到 localStorage
- [ ] 中央区上下比例持久化
- [ ] 上次使用的飞轮类型和材料记住

## v0.3.0 — 扩展飞轮类型与高级分析

**主题**：完善全部 5 种飞轮类型的完整 UI 支持，增加高级分析功能。

### 1. 完善飞轮类型 UI
- [ ] **等强度轮 (ConstantStrength)**：专用参数输入（渐变厚度曲线参数）
- [ ] **多层复合轮 (MultiLayerComposite)**：层编辑器 UI（添加/删除层、每层材料和厚度）
- [ ] 各类型截面预览 SVG 适配（锥形轮廓、多层轮廓渲染）

### 2. 疲劳寿命估算
- [ ] 基于应力幅值和材料 S-N 曲线估算疲劳寿命
- [ ] 显示预期循环次数和对应使用年限
- [ ] 疲劳安全裕度可视化

### 3. 温度场影响（可选）
- [ ] 考虑离心应力与热应力的耦合
- [ ] 温度对材料性能的影响（屈服强度下降）
- [ ] 温度-应力分布图

### 4. 参数敏感性分析
- [ ] 选择目标参数（如最大应力）和变化参数（如外径）
- [ ] 自动扫描参数范围生成灵敏度曲线
- [ ] 识别对安全性影响最大的设计参数

## 优先级排序原则

1. **用户价值优先**：直接影响核心工作流的功能排在前面
2. **依赖关系优先**：被后续功能依赖的基础能力先做
3. **风险可控优先**：技术风险高的功能提前验证（如 3D 渲染性能）
4. **小步迭代**：每个版本聚焦 2-3 个核心功能，避免范围过大
