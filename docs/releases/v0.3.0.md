# FlyForge v0.3.0

FlyForge v0.3.0 以“工程可用度提升”为核心目标，围绕多层飞轮与疲劳安全评估能力，完成了 6 项重点能力落地，并对前端预览与图表系统做了结构性重构，使得 FlyForge 更接近一个“可被工程人员日常使用的飞轮方案对比工具”。

本版本同时修复了若干会影响稳定性与构建流程的问题，包括曲线视图异常、敏感性计算崩溃、Rust 构建失败等关键缺陷。

## 版本定位

v0.3.0 的重点不再是“补全基础形状”，而是开始进入工程评估层：

- 飞轮截面不再只是“好看”，而是尽量对齐 ANSYS 中多层环的实际结构定义
- 评估指标不再只是“数值存在”，而是开始具备疲劳、温度、敏感性等更贴近工程设计的分析维度
- 可视化不再只是“画一张图”，而是逐步形成多视图联动和工程解释能力

---

## 新增能力

### 1. 多层复合轮结构重建

对 `MultiLayerComposite` 的截面计算进行了重构。早期版本中多层环更多停留在模型抽象层面；新版中每一层的厚度与半径约束将直接参与转动惯量计算与截面预览，使结果更贴近 ANSYS 中的多层环模型。

新增内容：
- 多层配置项输入（`layer_configs`）
- 每层独立厚度、外径控制
- 预设方案支持
- 截面预览同步显示分层结构

### 2. 疲劳寿命初值评估

新增基于 S-N 曲线与 Goodman 修正的疲劳寿命初值评估模块。该模块目前主要用于方案对比和早期设计筛选，尚不替代专业疲劳校核。

新增内容：
- S-N 疲劳寿命计算内核
- Goodman 应力比修正
- 疲劳寿命结果卡片
- 疲劳安全系数展示

### 3. 参数敏感性分析

新增参数敏感性分析模块，支持对 7 类核心设计参数进行 8 类关键指标扫描，帮助设计者快速识别“哪些参数对结果更敏感”。

支持扫描参数：
- 外径、内径、厚度、轮毂径、轮毂厚、额定转速、最大转速

支持扫描指标：
- 最大应力、质量、转动惯量、额定储能、可用能量、比能量、屈服安全系数、疲劳安全系数

### 4. 温度场影响评估

新增温度场相关估算能力，用于评估工作温度升高后对材料屈服强度、热应力水平的影响。

新增内容：
- 工作温度输入
- 温度对屈服强度的降额处理
- 热应力估算
- 热应力可视化曲线

### 5. S-N 曲线可视化

新增 S-N 曲线图，帮助用户更直观理解材料疲劳特性、疲劳极限与当前工况点位置。

新增内容：
- S-N 曲线绘制
- 疲劳极限参考线
- 当前工作应力点标注
- 曲线与十字线交互查询

### 6. 截面预览与剖面图重构

本版本对中心预览区进行了较大重构：

- 剖面图从“嵌入圆形预览内部”改为“独立固定高度条”
- 圆形预览可自由缩放/拖拽
- 剖面图支持水平同步圆形视图
- 剖面图支持独立拖拽、独立重置对齐
- 剖面图显示/隐藏开关可持久化

这一改动主要解决旧方案中剖面图容易被裁剪、视图冲突、布局不稳定的问题。

## 问题修复

本版本修复了以下关键问题：

### 预览与剖面图
- 修复剖面图被父容器裁剪导致不显示的问题
- 修复截面预览与剖面图重叠、布局冲突问题
- 修复锥形盘/等强度轮/多层复合轮剖面图缺失问题
- 修复拖拽分隔线不响应的问题
- 修复剖面图重置后不与主视图对齐的问题

### 灵敏度与计算稳定性
- 修复切换“内径 → 最大应力”时程序卡死或直接崩溃的问题
- 修复空心盘在 r=0 处出现 0/0 NaN 传播的问题
- 修复 `partial_cmp` 遇到 NaN 后触发 panic 的问题
- 为灵敏度计算增加防崩溃保护，避免单点失败影响整体页面

### 构建与发布流程
- 修复 `tauri build` 时 fold 结构解构错误导致编译失败的问题
- 修复 `sigma_vm` 移动后再次借用导致编译失败的问题
- 移除未使用导入，恢复构建链路稳定

### 前端稳定性
- 修复 S-N 曲线切换后页面崩溃的问题
- 修复 `xFormat` / `yFormat` 遇到 `null` 或 `NaN` 时报错的问题
- 修复曲线数据中异常点未过滤导致渲染失败的问题
- 修复灵敏度扫描结果为空或含非法值时前端报错的问题

## 使用建议

- 疲劳、温度场、敏感性能力目前更适合“方案比较”和“趋势判断”，不适合作为最终工程放行依据
- 多层复合轮建议优先使用“预设方案 + 手动微调”方式输入，避免层间几何约束不合理
- 如需进行更严格的结构与疲劳评估，建议同步使用 ANSYS、FEA 等专业工具复核

---

## 已知限制

- S-N 曲线与疲劳计算仍以简化工程估算模型为主
- 部分材料温度模型使用经验降额方式，尚未完全覆盖复杂合金体系
- 目前未提供完整 release 二进制上传，如需运行请使用源码构建

---

## 后续版本展望

v0.3.0 之后，FlyForge 将继续向“设计对比与工程评估工具”方向演进，后续优先方向包括：

1. 多方案对比视图
2. 结果导出增强（报告 / 截图 / 曲线导出）
3. 材料库增强与温度特性进一步建模
4. 参数边界约束提示与设计建议
5. 疲劳模型进一步细化
