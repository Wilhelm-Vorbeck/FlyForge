# FlyForge v0.4.0 开发计划

> 主题：**设计对比与工程报告输出**

v0.4.0 的核心目标是从"单方案评估工具"升级为"多方案对比与报告输出工具"，让工程人员能够一次性对比多个飞轮设计方案，并将结果导出为可交付的工程报告。

---

## D1 — 多方案对比视图

**目标**：支持同时查看 2~4 个飞轮方案的计算结果，横向对比关键指标。

- 方案列表管理（保存/删除/重命名当前方案）
- 对比视图面板：表格形式展示各方案的质量、惯量、最大应力、安全系数、储能、比能量
- 对比雷达图：多维度指标可视化对比
- 方案数据存储在 localStorage，跨会话保持

**涉及模块**：
- `src/store/` — 新增方案管理状态
- `src/components/` — 新增 ComparePanel / RadarChart 组件
- `src-tauri/src/lib.rs` — 无 Rust 改动

---

## D2 — 工程报告导出

**目标**：一键生成包含截面图、应力曲线、关键指标的工程报告。

- 导出格式：HTML 报告（自包含，可直接浏览器打开）
- 报告内容：
  - 设计参数汇总表
  - 截面图 SVG 嵌入
  - 应力分布曲线（SVG 内嵌）
  - 安全评估结论
  - 生成时间戳
- 报告模板引擎（Handlebars 或纯 Rust 模板）

**涉及模块**：
- `crates/flyforge-core/src/export.rs` — 新增 HTML 报告生成
- `src-tauri/src/lib.rs` — 新增 `export_report` Tauri 命令
- `src/services/api.ts` — 新增 `exportReport` API
- `src/components/ResultsPanel.tsx` — 新增"导出报告"按钮

---

## D3 — 参数边界约束与智能提示

**目标**：当用户输入的参数组合可能导致计算失败或结果不合理时，主动提示并建议调整。

- 内径 ≥ 外径时红色高亮 + 提示文本
- 轮毂径 > 外径时警告
- 厚度为 0 或负数时阻止计算
- 安全系数 < 1.0 时结果面板醒目警告（已有，增强视觉）
- 参数边界智能建议（如内径推荐范围 = 外径的 10%~50%）

**涉及模块**：
- `src/components/NumberInput.tsx` — 增加 warning/error 状态
- `src/components/tabs/` — 各 Tab 增加参数合理性提示
- `src/utils/validation.ts` — 新增前端参数校验工具

---

## D4 — 材料库增强

**目标**：扩展材料属性维度，支持更精确的工程计算。

- 材料新增属性：泊松比、屈服强度温度系数、疲劳极限温度系数
- 材料选择面板增强：显示材料关键属性摘要
- 自定义材料编辑器：支持用户添加自定义材料并持久化
- 材料对比功能：并排显示两种材料的 S-N 曲线

**涉及模块**：
- `crates/flyforge-core/src/types.rs` — Material 结构体扩展
- `src/types/index.ts` — TS 类型同步
- `src/components/tabs/MaterialTab.tsx` — 材料面板增强

---

## D5 — 疲劳模型细化

**目标**：提升疲劳评估的工程可信度。

- 温度对疲劳极限的影响（已有温度降额，扩展到疲劳极限）
- 平均应力修正支持 Gerber 准则（除 Goodman 外）
- 疲劳寿命结果增加置信度说明
- 多种修正准则可选（Goodman / Gerber / Soderberg）

**涉及模块**：
- `crates/flyforge-core/src/fatigue.rs` — 新增 Gerber/Soderberg 修正
- `crates/flyforge-core/src/thermal.rs` — 温度对疲劳极限影响
- `src-tauri/src/lib.rs` — 命令参数扩展
- `src/components/ResultsPanel.tsx` — 疲劳准则选择 UI

---

## D6 — UI/UX 优化

**目标**：提升日常使用体验。

- 键盘快捷键支持（Ctrl+S 保存参数、Ctrl+E 导出、Ctrl+Z 撤销参数修改）
- 参数修改历史（最近 10 步撤销/重做）
- 结果面板数值动画（数值变化时平滑过渡）
- 深色/浅色主题切换（可选，低优先级）

---

## 开发顺序建议

| 顺序 | 模块 | 预估工作量 | 依赖 |
|------|------|-----------|------|
| 1 | D1 多方案对比 | 中 | 无 |
| 2 | D3 参数边界约束 | 小 | 无 |
| 3 | D4 材料库增强 | 小 | 无 |
| 4 | D5 疲劳模型细化 | 小~中 | D4 |
| 5 | D2 工程报告导出 | 中 | D1 |
| 6 | D6 UI/UX 优化 | 小 | 无 |

D1/D3/D4 可并行开发；D5 依赖 D4 的材料属性扩展；D2 在 D1 完成后效果最佳。
