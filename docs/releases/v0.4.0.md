# FlyForge v0.4.0

FlyForge v0.4.0 以**"设计对比与工程报告输出"**为核心主题，将 FlyForge 从"单方案评估工具"升级为"多方案对比与报告输出工具"。本版本实现了 6 大功能模块，覆盖方案对比、参数约束、材料增强、疲劳细化、报告导出与交互优化，使工程人员能够一次性对比多个飞轮设计方案，并将结果导出为可交付的工程报告。

---

## 版本定位

v0.4.0 的重点是"设计决策支持"：

- 不再只是"算一个方案"，而是能够"对比多个方案"并直观看出优劣
- 不再只是"看结果"，而是能够"导出报告"用于工程交付
- 不再只是"输入参数"，而是有"边界约束提示"防止错误输入
- 不再只是"Goodman 一种准则"，而是支持 Gerber/Soderberg 多种疲劳修正

---

## 新增能力

### 1. 多方案对比视图 (D1)

支持同时保存、管理和对比 2~4 个飞轮方案，帮助设计者快速筛选最优设计。

新增内容：
- 方案列表管理：保存当前方案（参数+材料+计算结果快照）、重命名、删除、清空
- 对比表格：横向展示各方案的质量、惯量、最大应力、安全系数、储能、比能量等关键指标
- 雷达图对比：6 维度（比能量/安全系数/储能/质量/惯量/应力）纯 SVG 多边形可视化对比
- 方案数据持久化：localStorage 跨会话保持
- 最多支持 4 个方案同时对比

### 2. 工程报告导出 (D2)

一键生成自包含 HTML 工程报告，可直接在浏览器中打开或邮件发送。

新增内容：
- 暗色主题 HTML 报告模板
- 设计参数汇总表（几何/材料/工况）
- 计算结果表（质量/惯量/应力/安全系数/储能/疲劳寿命）
- 内联 SVG 应力分布曲线图
- 安全评估结论与建议
- 生成时间戳
- 支持自定义保存路径（通过 Tauri 文件对话框）

### 3. 参数边界约束与智能提示 (D3)

实时校验参数输入，防止不合理组合导致计算失败或结果失真。

新增内容：
- 几何参数校验：内径 ≥ 外径时 error（红色）、轮毂径 > 外径时 warning（黄色）
- 厚度校验：≤ 0 时 error
- 工况参数校验：转速顺序（最小 ≤ 额定 ≤ 最大）、安全系数范围
- NumberInput 组件支持 error/warning 状态（边框颜色 + 提示文本）
- 参数建议提示（如内径推荐范围 = 外径的 10%~50%）

### 4. 材料库增强 (D4)

扩展材料属性维度，支持自定义材料直接参与后端计算。

新增内容：
- 自定义材料编辑器：7 个属性字段（密度/杨氏模量/泊松比/屈服强度/抗拉强度/疲劳极限/热膨胀系数）
- 自定义材料持久化存储（localStorage）
- `material_override` 机制：自定义材料直接传入 Rust 后端参与计算，不依赖预设材料 ID
- 材料属性摘要显示

### 5. 疲劳模型细化 (D5)

提升疲劳评估的工程可信度，支持多种平均应力修正准则。

新增内容：
- `FatigueCriterion` 枚举：Goodman / Gerber / Soderberg 三种准则
- Gerber 抛物线修正：适用于脆性材料或高应力比场景
- Soderberg 直线修正：更保守的评估方式
- 结果面板下拉选择疲劳准则，实时重新计算疲劳寿命
- 温度对疲劳极限的降额处理（基于已有温度模型扩展）

### 6. UI/UX 优化 (D6)

提升日常使用的操作效率与体验。

新增内容：
- **参数撤销/重做**：`Ctrl+Z` / `Ctrl+Y`，最多 20 步历史记录
- **键盘快捷键**：`Ctrl+L` 切换左栏显示/隐藏、`Ctrl+R` 切换右栏显示/隐藏
- **布局状态记忆**：侧边栏折叠状态、中央区上下比例、剖面图显示/隐藏、飞轮类型和材料选择在重启后自动恢复
- 参数修改历史管理器（`ParamHistory` 类，滚动窗口实现）

## 使用建议

- **多方案对比**：建议先保存"基准方案"，再微调参数保存"对比方案"，最后使用对比表格+雷达图筛选最优设计
- **工程报告**：报告为自包含 HTML，可直接用浏览器打开，也可嵌入邮件或 Wiki
- **疲劳准则选择**：Goodman 为通用默认；Gerber 适用于脆性材料；Soderberg 适用于要求更保守评估的场景
- **自定义材料**：自定义材料会覆盖预设材料参与计算，确保所有 7 个属性字段填写正确

---

## 已知限制

- 多方案对比最多支持 4 个方案同时对比
- 工程报告目前仅支持 HTML 格式，暂不支持 PDF/Word 导出
- 参数撤销历史最多保存 20 步
- 疲劳模型仍以简化工程估算为主，关键设计建议配合专业疲劳分析工具复核

## 后续版本展望

v0.4.0 之后，FlyForge 将继续向"工程级设计工具"方向演进，后续优先方向包括：

1. 3D 飞轮模型预览与导出（STL/OBJ）
2. 参数优化与自动寻优（给定约束下自动搜索最优尺寸）
3. 更多飞轮类型（变厚度盘、辐条轮等）
4. 数据持久化增强（数据库/项目文件管理）
5. 多语言支持
