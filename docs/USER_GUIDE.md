## 一、应用简介

FlyForge 是一款专业的飞轮设计及运动学模拟工具，专为机械工程师和储能系统设计师打造。应用提供完整的飞轮设计分析功能，包括：

- **几何建模**：支持5种飞轮结构类型
- **应力分析**：基于Lame解的精确应力计算
- **安全评估**：屈服/疲劳/破裂安全系数分析
- **储能计算**：能量密度、可用能量窗口分析
- **动力学模拟**：RK4 ODE求解器模拟加速/减速过程
- **不平衡分析**：ISO 1940标准平衡等级计算

### 3.3 分析功能

#### 应力分析
- **Lame解**：精确计算旋转圆盘的径向和周向应力分布
- **von Mises等效应力**：用于屈服准则评估
- **边界条件验证**：自动检查自由表面应力为零

#### 安全评估
- **屈服安全系数**：`n_yield = σ_yield / σvm_max`
- **疲劳安全系数**：`n_fatigue = σ_fat_limit / σvm_max`
- **破裂转速**：`rpm_burst = rpm_rated × √(σ_ult / avg_hoop)`
- **临界转速**：基于屈服极限的理论临界转速

#### 储能分析
- **额定储能**：`E = 0.5 × J × ω²`
- **可用能量窗口**：`E_usable = 0.5 × J × (ω_max² - ω_min²)`
- **比能量密度**：`u_m = E / m` (Wh/kg)
- **速度波动系数**：`Cs = (ω_max - ω_min) / ω_avg`

#### 动力学模拟
- **RK4 ODE求解器**：四阶Runge-Kutta方法
- **转矩函数**：支持常数、线性、粘性摩擦等类型
- **加速/减速过程**：模拟充放电过程转速变化

#### 不平衡分析
- **ISO 1940标准**：`U_per = 9549 × G × m / n` (g·mm)
- **平衡等级**：G0.4至G4000（飞轮典型G6.3）
- **轴承动载荷**：`F_bearing ≈ U × ω² / (2×g)`

## 四、使用指南

### 4.1 快速入门

1. **选择飞轮类型**：在左侧侧边栏底部选择飞轮结构类型
2. **设置几何参数**：在"几何参数"Tab中输入外径、内径、厚度等
3. **选择材料**：在"材料选择"Tab中从12种内置材料中选择
4. **配置工况**：在"工况参数"Tab中设置转速范围和安全系数
5. **运行仿真**：点击"运行仿真"按钮执行计算
6. **查看结果**：在"计算结果"面板查看关键指标
7. **可视化分析**：切换到"可视化"Tab查看应力分布、转速曲线等图表
8. **导出数据**：使用"导出结果"面板导出CSV/JSON/SVG格式

### 4.2 预设方案

应用提供6种预设配置，可快速加载典型设计参数：

| 预设名称 | 飞轮类型 | 典型应用 |
|----------|----------|----------|
| 标准钢制飞轮 | 环形轮 | 通用工业储能 |
| 高速碳纤维飞轮 | 环形轮 | 高性能储能系统 |
| 低速实心飞轮 | 实心圆盘 | 教学演示、低速储能 |
| 锥形优化飞轮 | 锥形盘 | 质量优化设计 |
| 多层重型飞轮 | 多层复合轮 | 重型工业储能 |
| 等强度轮廓飞轮 | 等强度轮 | 高速储能、应力优化 |

### 4.3 自动计算

开启"参数变化时自动计算"开关后，每次修改参数都会自动触发仿真计算（500ms防抖），方便实时调整设计参数。

### 4.4 导出功能

| 格式 | 说明 | 用途 |
|------|------|------|
| CSV | 表格数据（参数、结果、应力分布、转速曲线） | Excel分析、数据存档 |
| JSON | 完整仿真结果（含所有原始数据） | 程序接口、数据交换 |
| SVG | 应力分布矢量图（含图例和坐标轴） | 报告插图、打印输出 |
| 参数配置 | 当前参数的JSON格式 | 参数存档、配置共享 |

### 4.5 国际化支持

应用支持中文/英文界面切换（开发中）。

## 七、常见问题

### Q1: 应用启动后白屏？
A: 尝试以管理员身份运行，或检查系统是否安装了WebView2运行时。

### Q2: 计算结果与预期不符？
A: 请检查：
- 单位是否正确（输入为mm，不是m）
- 转速是否合理（rpm_max > rpm_rated > rpm_min）
- 内径是否小于外径（r_i < r_o）
- 安全系数是否大于1.0

### Q3: 导出的CSV文件如何用Excel打开？
A: 在Excel中选择"数据"→"从文本/CSV"，选择CSV文件，分隔符选择"逗号"。

### Q4: 如何自定义材料？
A: 当前版本暂不支持自定义材料，后续版本将添加此功能。

### Q5: 应用占用多少系统资源？
A: FlyForge 是轻量级应用，典型运行时占用约50MB内存，CPU使用率很低（仅计算时短暂升高）。
