# Changelog

## [0.1.0] - 2026-06-18

### 新增功能

- **飞轮参数设计**
  - 支持环形轮（等厚度）和锥形轮两种飞轮类型
  - 几何尺寸输入：外径 Ro、内径 Ri、轮缘厚度、轮毂径、轮毂厚度
  - 材料库：12 种预设材料（合金钢、铝合金、钛合金、碳纤维、铸铁等）
  - 工况参数：额定/最大/最小转速、安全系数（屈服/疲劳/破裂）、离散点数量
  - 预设方案快速加载（8 种典型飞轮配置）

- **飞轮截面预览**
  - SVG 实时渲染飞轮截面轮廓
  - 支持滚轮缩放和鼠标拖拽平移
  - 网格背景 + 坐标轴
  - 颜色图例标注（外径/内径/轮毂）
  - 重置视图按钮

- **可视化图表**
  - 应力分布图：von Mises、周向应力、径向应力随半径变化曲线
  - 转速曲线：启动/制动过程转速-时间曲线
  - 能量柱状图：额定/最大/可用储能对比
  - 十字准星跟踪 + 实时数据浮动框
  - 滚轮缩放 + 拖拽平移 + 重置按钮
  - 特殊极值点标注

- **计算结果面板**
  - 质量、转动惯量
  - 最大应力及位置
  - 屈服/疲劳安全系数
  - 安全破裂转速
  - 额定/可用/最大储能、比能量
  - 安全评估状态（通过/未通过）

- **数据导出**
  - CSV 格式导出仿真数据
  - JSON 格式导出完整结果
  - SVG 格式导出图表
  - 参数 JSON 导出
  - 支持自定义保存路径（File System Access API）

- **UI/UX**
  - 黑绿科技配色主题
  - 左右侧边栏可折叠
  - 中央区分隔线可拖拽调整上下比例
  - 数字输入框支持长按快速微调
  - 扁平化飞轮应用图标
  - 实时计算（300ms 防抖）

### 技术栈

- Tauri v2 (Rust + WebView)
- SolidJS + TypeScript
- Tailwind CSS
- Vite
